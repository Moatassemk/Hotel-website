

# Hotel DJERBA Website

This is the README file for the Hotel DJERBA website code. Below, you will find an overview of the HTML and CSS code provided.

## Overview

The Hotel DJERBA website is a simple web page designed to promote a hotel. It includes a navigation bar, a large showcase section, information about the hotel, and a section showcasing various features and amenities of the hotel. The page concludes with a footer containing social media links.

## HTML Structure

The HTML structure of the code is as follows:

1. `<!DOCTYPE html>`: The document type declaration for HTML5.
2. `<html lang="en">`: The root element of the HTML document with a specified language.
3. `<head>`: The head section containing meta tags, links to external stylesheets (Font Awesome and Bootstrap), and the page title.
4. `<style>`: Internal CSS styles for the webpage.
5. `<body>`: The main content of the webpage.

### Navigation

- `<nav id="navbar">`: The navigation bar at the top of the page.

### Showcase

- `<div id="showcase">`: The large banner section at the top of the page.

### Hotel Information

- `<section id="home-info">`: A section containing information about the hotel.

### Features

- `<section id="features">`: A section highlighting various features and amenities of the hotel.

### Footer

- `<footer>`: The footer section at the bottom of the page with social media links.

## CSS Styles

The code includes CSS styles for various elements and classes used in the HTML. It defines the appearance and layout of the webpage, including colors, fonts, and responsiveness.

## Usage

This code is a static HTML page, and you can use it as a template for a hotel website. You can customize the content, images, and styles to suit your specific needs.

## External Dependencies

The code includes external dependencies for styling:

- Font Awesome (for icons)
- Bootstrap 4.5.2 (for styling)

You should ensure that these external dependencies are available for the webpage to display correctly.
